# Actividad 2.6 - Exploración de JavaScript Backend con Cheerio

este proyecto consiste en el desarrollo de una aplicacion backend utilizando node.js,express y cherio

el objetivo El objetivo es procesar contenido HTML desde un archivo local, extraer información mediante selectores CSS y entregar los resultados de forma estructurada a través de un endpoint.

La aplicación implementa una arquitectura por capas utilizando routes, controllers y services, además de validaciones básicas y manejo de errores para asegurar el correcto funcionamiento del sistema.
